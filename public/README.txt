Dépose ici l'image de couverture (le médaillon de l'École du Loup).

- Nomme le fichier exactement comme la valeur de `coverImage` dans
  src/gameData.js (par défaut : tw_chair_bg.svg).
- Formats acceptés : .png (transparence recommandée), .svg, .jpg, .webp.
- Tout fichier placé dans /public est servi à la racine du site.

Si aucune image n'est trouvée, la couverture s'affiche simplement sans médaillon
(aucune erreur). Mets coverImage à '' dans gameData.js pour la masquer volontairement.
